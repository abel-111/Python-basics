# Coffee Machine Simulator

A console-based Python project that simulates a coffee vending machine.

## Features

- Order Espresso, Latte, and Cappuccino
- Coin-based payment system
- Resource management (water, milk, coffee)
- Change calculation
- Resource reporting
- Machine shutdown option

## Concepts Used

- Python Dictionaries
- Functions
- Loops
- Conditional Statements
- User Input Handling

## Technologies

- Python
