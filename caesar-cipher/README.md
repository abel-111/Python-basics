# Caesar Cipher

A simple command-line tool to encrypt and decrypt messages using the Caesar Cipher algorithm.

## How It Works

The Caesar Cipher shifts each letter in a message by a fixed number of positions in the alphabet.  
For example, with a shift of 3: `a → d`, `b → e`, `z → c`.

## How to Run

```bash
python caesar.py
```

## Usage

1. Choose `encode` to encrypt or `decode` to decrypt
2. Type your message
3. Enter a shift amount (any number)
4. Optionally repeat with a new message
