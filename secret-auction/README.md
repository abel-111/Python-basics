# Secret Auction

A command-line auction program where multiple bidders can enter their bids privately.
After all bids are placed, the highest bidder is revealed.

## How It Works

1. Each bidder enters their name and bid amount
2. The screen clears between bidders to keep bids secret
3. Once all bidders are done, the winner and winning bid are announced

## Usage

```bash
python secret_auction.py
```

## Example
Bidding Program!
Enter your name: Abel
Enter your bid: 500
Add another bidder? (yes/no): yes

Enter your name: John
Enter your bid: 750
Add another bidder? (yes/no): no
