# Python Problems Collection

## 📌 Overview
This repository contains solutions to basic Python programming problems. The notebook covers beginner-friendly exercises focused on core Python concepts such as variables, input/output, loops, conditionals, lists, and pattern printing.

## 🚀 Topics Covered

### 1. Area of a Circle
- Accepts radius as input
- Calculates and displays the area of a circle

### 2. Student Information
- Takes student details as input
- Displays name, age, and marks

### 3. Find Maximum Number in a List
- Uses a loop to determine the largest element in a list

### 4. Current Number, Previous Number, and Sum
- Prints current number, previous number, and their sum

### 5. Numbers Divisible by 5
- Iterates through a list
- Displays numbers divisible by 5

### 6. Prime Number Checker
- Checks whether a given number is prime or not

### 7. Reverse a List
- Reverses a list using a loop

### 8. Star Pattern Printing
- Prints a simple increasing star pattern

### 9. Largest of Three Numbers
- Finds the largest among three user-input numbers

### 10. Pyramid Pattern
- Prints a full star pyramid pattern
