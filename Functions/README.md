# Basic Python Functions

A collection of beginner-friendly Python functions created to practice core programming concepts such as functions, loops, conditional statements, list operations, and string manipulation.

## Features

- Find the maximum of two numbers
- Check whether a number is even or odd
- Calculate the factorial of a number
- Find the sum of elements in a list
- Calculate simple interest
- Identify whether a number is positive, negative, or zero
- Generate the Fibonacci sequence
- Find the second largest element in a list
- Count vowels in a string

## Concepts Covered

- Function creation and usage
- Conditional statements (`if`, `elif`, `else`)
- Loops (`for`, `while`)
- List operations
- String handling
- Mathematical calculations
- Return values and parameters

## Technologies Used

- Python 3
- Google Colab / Jupyter Notebook

## How to Run

1. Open the notebook in Google Colab or Jupyter Notebook.
2. Run each cell sequentially.
3. Observe the outputs for each function.

## Learning Objective

This notebook is intended for Python beginners who want hands-on practice with fundamental programming concepts and problem-solving techniques.
